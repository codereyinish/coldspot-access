# ColdSpot — Request Access

**ColdSpot** is an educational systems-networking project: it routes *all* of a
Mac's traffic through a paired iPhone out to a **self-hosted exit server you own**
— demonstrating Layer-3 packet capture, a userspace TCP/IP stack, a reverse
tunnel, and authenticated SOCKS5-over-TLS.

The code lives in a **private** repository. Want to experiment with it?

## 🔑 Request access

👉 **[Open a "Request access" issue](../../issues/new?template=request-access.yml)**
and include your **GitHub username**.

Once approved, you'll be added as a **read-only collaborator** on the private repo
(you can clone and run it, but not change it). I review requests manually — so
I'll see who's asking. 🙂
